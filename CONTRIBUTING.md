> ## NOTE, usage DISCOURAGED: use [elm-keysdict](https://github.com/indique/elm-keysdict), which allows for multi-directional access

### Summary Guide

commits look like this:
> do it: details | other action: details

#### Category
| mark      | means   |
| :-------  | :------ |
| +: ...    | add     |
| -: ...    | remove  |
| +-: ...   | change  |
| ↻: ...    | update  |
| ✓: ...    | correct |
| -<: ...   | split   |
| >-: ...   | merge   |

#### Others
| mark        | means         |
| :---------- | :------------ |
| #...        | see           |
| ! ...       | **attention** |
